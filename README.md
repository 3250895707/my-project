# Privacy-preserving distributed k-WTA algorithm

## 项目说明

本项目是论文《Privacy-preserving distributed k-WTA algorithm based on state combination》的实验代码，实现了基于状态组合的隐私保护分布式k-WTA算法。

## 作者

Zishuo Liu, Yinyan Zhang

## 单位

暨南大学网络空间安全学院（College of Cyber Security, Jinan University）

## 文件说明

- `实验一.zip` - 4节点正确性验证
- `实验二.zip` - 参数影响实验
- `实验三.zip` - 初始值影响实验
- `实验四.zip` - 80节点大规模实验
- `4节点拓扑图.zip` - 通信拓扑图
